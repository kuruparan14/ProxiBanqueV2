Etape de lancement de l'éxecutable ProxiBanquev2.war

1- Copier le fichier ProxiBanquev2.war dans le dossier webapps de repertoire apache-tomcat-7.0.90 
2- Lancer le service "tomcat" en double cliquant sur le fichier "startup.bat" dans le dossier "bin" de repertoire apache-tomcat-7.0.90
3- ouvrir un navigateur web et tapper la commande localhost:8080/proxibanquev2
4- Pour arreter le service tomcat, double cliquer sur le fichier "shutdown.bat" dans le dossier "bin" de repertoire apache-tomcat-7.0.90
